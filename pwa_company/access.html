Transparency internal console

  • Press F12 to enter console

Command

Press F12 to enter console (Use google chrome)

1. memList()

check all members data on server.


2. cartList()

check all cartList data on server.


3. setZero(cartId, stat)

clear data to cart (cartId) with (stat). / e.g. setZero("000000", 0)


4. setCartStatus(cartId, stat)

Set (cartId) to (status) / e.g. setCartStatus("000000", -1)


5. setTS(message)

Write Thinkspeak value to (message) / e.g. setTS("4gJeL,3,Ti3h1,2,Z73Bs,3")
